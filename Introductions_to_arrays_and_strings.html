Introductions to arrays and strings


An array represents a set of similar values. The array declaration is as follows:

variable_type array_name [array_length]

After the type of the variable is the name of the array, and then in square brackets its size. For example, define an array of 4 numbers:

1 int numbers[4];

Using the element numbers of the array, which are called indexes, we can refer to individual elements. For example:

Indexes start from zero, therefore, to access the first element, you must use the numbers [0] expression.

We can also immediately declare and initialize the array with values. For this array variable, a set of values ​​separated by commas in curly brackets is assigned:

1

2		 int numbers[4] = { 1, 2, 3, 5 };

printf("numbers[2] = %d", numbers[2]); // 3

When initializing the array, you can not explicitly specify its length, in this case the length of the array will be calculated based on the number of its elements:

1 int numbers[] = { 1, 2, 3, 5 };

Using circular constructs, you can iterate over an array. It is not necessary to initialize the array with values ​​for all its elements.

Multidimensional Arrays

Arrays can be multidimensional. Elements of such arrays themselves, in turn, are arrays, in which elements can also be arrays. For example, define a two-dimensional array of numbers:

1 int numbers[3][2] = { {1, 2}, {4, 5}, {7, 8} };

To access the elements of a nested array, two indexes are required:

1

2		 int numbers[3][2] = { {1, 2}, {4, 5}, {7, 8} };

printf("numbers[1][0] = %d \n", numbers[1][0]); // 4

Strings

Arrays of numbers were considered above, but with arrays of other data types everything will be the same. But it is worthwhile to dwell on arrays of characters. Different programming languages ​​have special data types for representing strings. In the C programming language, arrays of characters are used to represent strings, because in fact a string is a set of characters. For example, define a line:

     return 0;

Strings are specified in double quotation marks. And if we need strings in the program, then we can just use arrays of characters.

But it is worth noting that in addition to the characters themselves, which are enclosed in double quotes, each line as the trailing character contains the character '\ 0' or a null character. Therefore, the string "Hello" will actually not be 5 characters, but 6.
