Functions

Passing parameters and returning the result


The function can take parameters. The parameters are listed in brackets after the function name have the following definition:

type parameter_name

For example, we define a factorial calculation function:

		#include <stdio.h>
 
		void factorial(int n)
		{
			int result = 1;
			for(int i=1;i<=n;i++)
			{
				result *= i;
			}
			printf("factorial of %d is equal %d \n", n, result);
		}
		 
		int main(void)
		{
			factorial(4);
			factorial(6);
			return 0;
		}

The factorial function takes one parameter, which has the name n and type int.

In the main method, this function is called twice, and each time a new value for parameter n is passed to it. The passed value for the parameter is also called an argument. As a result, the console function is the factorials of numbers 4 and 6

If several parameters are needed, they are listed with a comma. For example, we write a program simulating a currency exchange:

		#include <stdio.h>
 
		void exchange(double currate, double sum)
		{
			double result = sum / currate;
			printf("result=%.2f \n", result);
		}
		 
		int main(void)
		{
			double rate;
			double sum;
			
			printf("Currency rate:");
			scanf("%lf", &rate);
		 
		 
			printf("Sum:");
			scanf("%lf", &sum);
		 
			exchange(rate, sum);
			 
			exchange(56.87, 2000);
			 
			return 0;
		}

Using the scanf () function, console input is applied to the rate and sum variables, then these variables are passed to the exchange function as arguments. Variables used as arguments do not have to be named exactly like function parameters. The correspondence between arguments and parameters is established by their location: the first argument passes the value for the first parameter, the second argument for the second parameter, and so on. And besides, here the second call to the exchange function takes numeric constants as arguments.

A function may return some specific value. For example, we modify the factorial function to return a calculated value:

		#include <stdio.h>
 
		int factorial(int n)
		{
			int result = 1;
			for(int i=1;i<=n;i++)
			{
				result *= i;
			}
			return result;
		}
		int main(void)
		{
			int arg,f;
			 
			arg = 4;
			f = factorial(arg);
			printf("factorial of %d is equal %d \n", arg, f);
			 
			arg = 6;
			f = factorial(arg);
			printf("factorial of %d is equal %d \n", arg, f);
			 
			return 0;
		}

The factorial function now has an int type as the return type. If the type of the function is different from void, then the function must return a value of this type. To return a value, the return statement is used, after which the return value is indicated.

In this case, the factorial function returns the value of the result variable, which contains the factorial of the number.

If the function returns a value, then when we call it, we can get this value in a variable

As with the parameters and arguments of a function, the return value must match the type of the function. If there is no such correspondence, the compiler tries to convert the return value to the type of the function:

		#include <stdio.h>
 
		int exchange(double rate, double sum)
		{
			return sum/rate;
		}
		 
		int main(void)
		{
			printf("%d \n", exchange(56.83, 24560));
			 
			return 0;
		}

In this case, the exchange function returns an object of type int, but the return value described by the expression sum / rate will be of type double, so in this case, the compiler will automatically perform the conversion.
