Input (output) data


Input to the console. Scanf function

The scanf () function can be used to enter data into the console. This function is defined in the stdio.h header file (in the same place as the printf function) and has the following formal definition:

1 scanf(форматная_строка, аргументы);

Both format_string and arguments to the scanf function are required. Format_String contains conversion specifications that define the input.

The specifier determines the type of input data:

  • % c: read one character
  • % d: read decimal integer
  • % i: reads an integer in any system (decimal, hexadecimal, octal)
  • % u: reads a positive integer
  • % e: read exponentially floating point number
  • % E: read exponentially floating-point number with exponent capital
  • % f: reads a floating-point number
  • % F: reads a floating point number
  • % g: read floating point number
  • % G: reads a floating point number
  • % o: reads an octal number
  • % x: reads a hexadecimal number
  • % X: reads a hexadecimal number
  • % s: read line
  • %%: read percent symbol

The asterisk * in the conversion specification allows you to skip typed characters when entering for the type specified through the specifier.

Field_width represents a positive integer that allows you to determine how many bytes will be taken into account when entering.

Modifiers allow you to specify the data type. In particular, there are the following modifiers:

  • h: to enter values ​​of type short int (% hd)
  • l: to enter values ​​of type long int (% ld) or double (% lf,% le)
  • L: to enter values ​​of type long double (% Lf,% Le)

The arguments to the scanf () function are the addresses of the variable that will receive the entered value. To get the address of a variable, an ampersand & is placed before its name. For example, if the variable is called age, then we can get its address using the & age expression.

You can enter several values ​​at once. In this case, a space is used as the delimiter.

Console output. Printf function

To print to the console in the C programming language, the printf () function is often used. But if there are no special problems with the output of a simple string, then the output of data of other types has its own peculiarities.

The formal definition of the printf function is as follows:

1 printf (format_string, argument_list);

In addition to standard text, we can pass special groups of characters called control sequences to a string. The most common ones are:

  • \ n: newline
  • \ t: tab
  • \ r: carriage return (cursor) to the beginning of the line
  • \\: backslash
  • \ ': single quote
  • \ ": double quote

Additionally, you can use flags that additionally allow you to control the formatting of the output:

  • +: if the displayed value has a sign (+ or -), then it is displayed. Without this flag, the sign is displayed only if the value is negative.
  • space: inserts a space in the place of the character in front of positive numbers
  • #: when used with the qualifiers "o", "x", "X", the value of the number is displayed with the preceding characters 0, 0x or 0X. When used with the qualifiers "f", "g", "G", the decimal point will be displayed, even if there is no fractional part in the number
