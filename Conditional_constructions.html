Conditional constructions


If construct

Conditional designs allow you to direct the program along one of the possible paths, depending on the condition. There are several operators in the C language that allow this.

The if statement checks if the condition is true, and if it is true, it executes a block of instructions. This statement has the following abbreviated form:

if (выражение_условия) инструкция;

An arithmetic, logical, or comparison operation can be used as condition_expression. And if this operation returns a nonzero value (that is, the expression is true), then the following statement is executed:

    if(x < 30) printf("x is less than 30 \n");

Two conditional if constructs are used here. They will check for more or less the value of the variable x than a specific value. In both cases, the printf function is executed as an instruction.

In the first case, x> 50, the condition is true, since the value of the variable x is really greater than 50, so this comparison operation will return 1, and therefore the next instruction will be executed.

In the second case, the comparison operation x <30 will return 0, since the condition is false, so the subsequent instruction will not be executed. Therefore, only one line will be displayed on the console

Switch construct

Another form of branching programs is the switch ... case construct. It has the following form:

    case constant_2: instructions_2;

    default: instructions;

}

After the switch keyword, the compared expression is in parentheses. The value of this expression is sequentially compared with the values ​​placed after the case statement. And if a match is found, a specific case block will be executed.

At the end of the switch construct, there may be a default block. It is optional and is executed if no matches were found in case blocks.

To avoid executing subsequent case / default blocks, a break statement is placed at the end of each block.

Ternary operator

The ternary operator ?: Reduces the definition of the simplest conditional constructions if and has the following form:

[first operand condition] ? [second operand] : [third operand]

The operator uses three operands at once. Depending on the condition, the ternary operator returns the second or third operand: if the condition is 1 (that is, true), then the second operand is returned; if the condition is 0 (i.e. false), then the third.
