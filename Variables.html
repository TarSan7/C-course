Basic concepts

Variables


Variables are used to store data in the C programming language. The variable can have type, name and value.Any variable must be defined before use. The variable definition syntax is as follows:

1.type_name variable_name;

For example, the simplest definition of a variable is:

1 int number;

Here we define the variable number, which is of type int. Since the definition of a variable is a separate expression, it is followed by a semicolon. A variable name can represent a sequence of Latin alphabet characters, numbers, and an underscore. The name must begin with either an alphabetic character or an underscore.

Often, compilers have a variable length limit, for example, for some compilers the variable name may not exceed 31 characters, for other compilers the constraint may be more stringent. It is also worth considering that C is a case-sensitive language, which means that the character register is of great importance. That is, the following code will define two different variables:

1

2		 int number;

int Number;

Therefore, the Number variable will not represent the same as the number variable. In addition, there are a number of reserved words in the language that cannot be used as a variable name. Not so many: auto, break, case, char, const, continue, default, do, double, else, enum, extern, float, for, goto, if, int, long, register, return, short, signed, sizeof , static, struct, switch, typedef, union, unsigned, void, volatile, while.

Each variable is assigned an indeterminate value. But in the future, we can assign it a specific meaning:

number = 2;
