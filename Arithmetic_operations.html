Arithmetic operations


The C programming language supports all basic arithmetic operations. These operations are performed on numbers. The values ​​that participate in the operation are called operands. List of operations available in C:

  • +

    The addition operation returns the sum of two numbers:

  • *

    The multiplication operation returns the product of two numbers:

    1 int c = 15 * 3;
  • /

    The operation of dividing two numbers:

    1

    2    		 int a = 15 / 3;

    float d = 34.0 / 3.0;

    When dividing, you should be careful, because if two integers are involved in the operation, the division result will be rounded to an integer, even if the result is assigned to the float variable

  • %

    Returns the result of dividing two numbers modulo (getting the remainder of the division):

    1 int a = 35 % 3;

Operations *, /,% have a higher priority than + and -.

With the help of parentheses, we can redefine the order of operations by making operations with a lower priority be performed before operations with a higher priority.

Also in C there are special unary operations on one number: ++ (increment) and - (decrement). Each of the operations has two varieties: prefix and postfix.
