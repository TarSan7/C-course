Basic concepts

Loops


Cycles allow you to perform one action many times, depending on a specific condition. C has the following types of loops:

  • for
  • while
  • do...while

For loop

The for loop has the following formal definition:

1

2

3

4		 for (expression_1; expression_2; expression_3)

{

    // loop body

}

expression_1 is executed once at the beginning of the execution of the cycle and represents the installation of the initial conditions, as a rule, this is the initialization of counters - special variables that are used to control the cycle.

expression_2 represents a condition under which a loop is executed. As a rule, a comparison operation is used as a condition, and if it returns a nonzero value (that is, the condition is true), the loop body is executed, and then expression_3 is evaluated.

expression_3 specifies a change in the parameters of the cycle, often there is an increase in the counters of the cycle by one.

Consider the standard for loop:

1

2

3

4

5

6

7

8

9		 #include <stdio.h>

int main(void)

{

    for (int i = 0; i < 9; i++);

    {

        printf("%d * %d = %d \n", i, i, i * i);

    }

     return 0;

}

The first part of the loop declaration - int i = 0 - creates and initializes the counter i. The counter need not represent an int. It can be another numerical type, for example, float. And before the loop is executed, its value will be 0. In this case, it is the same as declaring a variable.

The second part is the condition under which the cycle will be executed. In this case, the loop will run until i reaches 9.

And the third part is the increment of the counter per unit. Again, we do not have to increase by one. You can reduce: i--. You can change to another value: i + = 2.

As a result, the cycle block will work 9 times until the value of i becomes equal to 9. And each time this value will increase by 1.

Each individual loop pass is called an iteration. That is, in the example above there were 9 iterations.

Do loop

In the do loop, the loop code is executed first, and then the condition is checked in the while statement. And while this condition is true, that is, it is not equal to 0, the cycle repeats.

While loop

Unlike the do loop, the while loop immediately checks the validity of a certain condition, and if the condition is true, that is, it is not equal to 0, then the loop code is executed

Continue and break statements

Sometimes it becomes necessary to exit the cycle before it is completed. In this case, you can use the break statement.

for ( ; ; )

{

    printf("%d * %d = %d \n", i, i, i * i);

    i++;

    if (i > 10) break;

}

Here, when the value of the variable i reaches 10, the loop is exited using the break statement.

Unlike the break statement, the continue statement jumps to the next iteration. For example, we need to calculate the sum of only odd numbers from a certain range. To find out if the number is even, we get the remainder of the integer division by 2, and if it is 0, then using the continue operator we go to the next iteration of the loop. And if the number is odd, then add it to the rest of the odd numbers.
